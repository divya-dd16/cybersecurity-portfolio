Objective
The objective of this activity was to identify and analyse security concepts implemented within the university campus environment to protect students, staff, infrastructure, and digital systems.

Security Implementations Observed
1. Microsoft Multi-Factor Authentication (MFA)
Implementation
Students must verify their identity through Microsoft authentication when accessing university systems such as email, learning portals, and internal resources.
Security Concept
Authentication and Identity Verification
Explanation
Multi-factor authentication requires users to verify their identity using more than one method, typically a password and a mobile authentication prompt. This reduces the risk of unauthorized access even if passwords are compromised.
2. Student Card Access Control
Implementation
Student cards are used to access university facilities such as libraries and buildings, particularly after regular hours.
Security Concept
Access Control
Explanation
RFID-enabled student cards ensure that only authorized individuals can access specific facilities. This limits unauthorized entry and improves campus safety.
3. Print Services Authentication (PrintMe System)
Implementation
Students must tap their student card on printers to release their print jobs after uploading documents.
Security Concept
Secure Authentication and Confidentiality
Explanation
This system prevents documents from being printed and left unattended. It ensures that only the owner of the document can release the print job.
4. CCTV Surveillance
Implementation
CCTV cameras are installed across campus in buildings, corridors, entrances, and outdoor areas.
Security Concept
Monitoring and Surveillance
Explanation
Surveillance systems help deter crime, monitor suspicious activities, and provide evidence in case of security incidents.
5. Lecture Recording Systems
Implementation
Lecture capture systems record classroom sessions and upload them to the university learning platform.
Security Concept
Accountability and Controlled Access
Explanation
Access to lecture recordings is restricted to enrolled students, ensuring that academic materials are shared securely within the learning environment.
8. Security Patrols
Implementation
Campus security officers patrol the campus using small electric vehicles similar to golf carts.
Security Concept
Physical Security and Threat Prevention
Explanation
Visible patrols help deter criminal activities and allow rapid response to incidents across campus.
9. Student Guild Membership Systems
Implementation
Guild membership systems verify student eligibility for services, discounts, and participation in university clubs.
Security Concept
Identity Verification and Authorization
Explanation
Membership systems ensure that only verified students can access student services and benefits.
